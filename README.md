## Завдання

### Завдання 1 — summarizeNumbers
Функція приймає масив чисел і повертає статистику: кількість, суму, кількість парних, максимум і категорію. Використано цикл for.

### Завдання 2 — processProducts
Функція обробляє масив товарів: фільтрує наявні, рахує суму, знаходить найдешевший, формує прайс-лист. Використано filter, reduce, map.

### Завдання 3 — createApiClient
Замикання з приватним лічильником запитів. Асинхронні fetch-запити через async/await з обробкою помилок try/catch.

### Завдання 4 — TodoList
Класи Task і TodoList. DOM-маніпуляції, делегування подій, перемикання стану завдань.
